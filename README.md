#  FraudShield AI - Transaction Fraud Detection System

FraudShield AI is a Machine Learning-based system designed to detect fraudulent financial transactions using behavioral and transactional patterns.

Built during a hackathon, the project demonstrates an end-to-end ML workflow including model training, preprocessing, and deployment through an interactive Streamlit application.

---

##  Features

- Real-time fraud prediction
- Machine Learning classification model
- Interactive Streamlit dashboard
- Transaction risk confidence score
- User-friendly prediction interface

---

##  Problem Statement

Financial fraud causes significant losses across digital platforms.  
FraudShield AI analyzes transaction attributes to classify whether a transaction is:
Genuine  
Fraudulent  

---

##  Tech Stack

- Python
- Scikit-learn
- Pandas & NumPy
- Streamlit
- Pickle

---

##  Machine Learning Workflow

1. Data preprocessing
2. Feature engineering
3. Model training using Random Forest
4. Model evaluation
5. Deployment using Streamlit
